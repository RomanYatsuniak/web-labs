HTML <b> element definiuje pogrubiony tekst, bez żadnego dodatkowego znaczenia.
HTML <strong> element definiuje pogrubiony tekst, z dodatkiem semantycznej „strong” znaczenie.

HTML <i> element definiuje kursywą tekst, bez żadnego dodatkowego znaczenia.
HTML <em> element definiuje podkreślony tekst, z dodatkiem semantycznego znaczenia.

Element <code> jest tag fraza. Definiuje ona kawałek kodu komputerowego.
Element <samp> jest tag fraza. Definiuje próbki wyjście z programu komputerowego.
